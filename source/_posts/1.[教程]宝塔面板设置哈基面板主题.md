# 前言

7月24日是宝塔运维节，宝塔推出双版本更新(即<b><i>v10.0.1</b></i>与 <b></i>v11.0.0</b></i> )在新版本中(<b><i>v11.0.0</b></i> )宝塔推出了全新UI界面，同时更新了 <b>自定义Logo图标</b>，<b>网站图标</b>，<b>侧边栏配色</b>，<b>主界面背景</b>，<b>登录界面背景</b> ,<b>图标</b> 的设置功能。在保证运维的便捷性，安全性的同时还允许用户自定义UI等。

# 正文

### 对宝塔此次更新的看法

先说一下对宝塔此次更新的看法

*     首先是关于 <b><i>v10.0.1</b></i> 版本，在 <b><i>v10.0.1</b></i> 版本中虽然没有更新新UI与自定义主界面背景，登录界面背景的功能，但推出了 <b>自定义Logo图标</b>，<b>网站图标</b>，<b>侧边栏配色</b> 功能，同时新推出了 <b>免费的多节点管理</b>，<b>负载均衡</b>，<b>MySQL</b>/<b>Redis主从复制</b>，<b>文件互传</b>，以及对负载均衡与主从复制的 <b>告警功能</b> 等。
*   其次是关于 <b><i>v11.0.0</b></i> 版本，在新安装中版本号为 <b><i>v11.0.0</b></i>，从 <b><i>v10.0.1</b></i> 更新的版本号为 <b><i>(稳定版) v10.0.2-LTS</b></i>，该版本推出了<b>全新的UI界面</b>相比之前的UI界面，新版本的UI界面更加圆润一些，当然也有部分用户认为新UI不如原UI，没有原来的UI好看，这个每个人都又不同的喜好，喜欢原UI的建议不要更新。在推出新UI的同时当然也推出了我们前言所说的 <b>自定义Logo图标</b>，<b>网站图标</b>，<b>侧边栏配色</b>，<b>主界面背景</b>，<b>登录界面背景</b>，<b>图标</b>的设置功能。

### 开始

上文有点跑题了，我的个人观点等就说到这里，下面让我们正式开始 <b>设置哈基面板主题</b>。

#### <b>安装/更新最新版宝塔面板</b>

*   安装最新版宝塔命令(请以宝塔官网[https://www.bt.cn](https://www.bt.cn)为主):

```bash
if [ -f /usr/bin/curl ];then curl -sSO https://download.bt.cn/install/install_latest.sh;else wget -O install_latest.sh https://download.bt.cn/install/install_latest.sh;fi;bash install_latest.sh ed8484bec
```

*   若<b>已安装宝塔</b>面板<u><b>请不要执行</b></u>上面的命令，已安装宝塔面板的用户<b>请单击面板右上角的更新按钮进行更新</b>。

Tips: 通过更新按钮更新的版本号为: <b><i>(稳定版) v10.0.2-LTS</b></i> (这里可能有不一样的，只要更新到了新UI版本即可)。 
[![]https://qn.cdn.yll14.cn/哈基BT/哈基面板_1.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_1.webp)新安装的面板版本号为: <b><i>v10.0.1</b></i>。
[![](https://qn.cdn.yll14.cn/哈基BT/哈基面板_2.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_2.webp)

#### 修改面板别名及左侧菜单标题

安装/更新好面板后登录面板单击 <b>左侧菜单栏的设置--></b> 设置界面的 <b>面板设置</b> ，将 <b>面板别名</b> 与 <b>左侧菜单标题</b> 修改为 <b>哈基面板</b> 记得一个一个改，改完一定要 <b>记得保存</b> 哦。 
[![](https://qn.cdn.yll14.cn/哈基BT/哈基面板_3.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_3.webp)

#### 修改主界面与登录页Logo及背景图

[哈基面板GIF下载](https://get.yll14.cn/%E5%93%88%E5%9F%BA%E9%9D%A2%E6%9D%BF/) 
在设置界面的<b>界面设置</b>，修改<b>Logo</b>与<b>网站图标</b>为<b>篮球耄耋</b>。在修改<b>网站图标</b>时修改<b>文件类型</b>为<b>所有文件</b>。 
[![](https://qn.cdn.yll14.cn/哈基BT/哈基面板_4.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_4.webp)

#### 修改主界面背景

开启<b>主界面背景图</b>功能上传<b>主界面背景图</b>，上传<b>蜘蛛耄耋</b>背景图，将<b>背景图透明度</b>与<b>内容透明度</b>设置为<b>2:1</b> (即<b>背景图透明度</b><b><i>100</b></i>，<b>内容透明度</b><b><i>50</b></i>)。记得保存哦！ 
[![](https://qn.cdn.yll14.cn/哈基BT/哈基面板_5.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_5.webp)

#### 修改登录页面Logo与登录背景图片

修改Logo为<b>篮球耄耋</b>，修改登录图片为<b>蜘蛛耄耋</b> 。 
[![](https://qn.cdn.yll14.cn/哈基BT/哈基面板_6.webp)](https://qn.cdn.yll14.cn/哈基BT/哈基面板_6.webp)
